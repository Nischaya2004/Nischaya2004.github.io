# Nischaya Barotha Interactive Portfolio Website

Interactive recruiter-focused static portfolio for GitHub Pages.

## Files

- `index.html` — main page with project tabs, mascot and recruiter-focused sections
- `styles.css` — responsive dark navy design with hover effects, pop buttons, card animations and mascot styling
- `script.js` — mobile navigation, scroll reveal, project filtering, cursor glow and hopping mascot

## How to deploy

1. Open `github.com/Nischaya2004/Nischaya2004.github.io`
2. Replace the existing `index.html`, `styles.css`, and `script.js`
3. Commit changes
4. Wait for GitHub Pages deployment
5. Open `https://nischaya2004.github.io/`

## Notes

Keep animations professional. The mascot is small and fixed at the bottom-right, so it adds personality without distracting from the recruiter content.
