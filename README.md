# Nischaya Barotha Balloon Buddy Portfolio

Interactive recruiter-focused portfolio for GitHub Pages.

## What is included

- Inflated balloon hover effects on navbar tabs, project filter tabs, buttons and links
- Blue theme hover color changes
- Project filtering tabs
- Scroll reveal animations
- Custom right-side interactive scroll ball
- Original CSS character that chases the scroll ball while you scroll or drag it
- No emoji mascot

## Deploy

Replace these files in `Nischaya2004.github.io`:

- `index.html`
- `styles.css`
- `script.js`

Commit and wait for GitHub Pages deployment.
